# secret-garden-server

## Quick start

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev`: start with `nodemon`
- `npm start`: start with `node`

## API

- `GET /`: basic service info
- `GET /api/health`: health check

## Environment variables

Copy `.env.example` to `.env` and update values if needed:

```bash
PORT=3000
```
