# 用户功能（精简版）

## 1. 范围说明
- 仅保留用户功能，不包含技术实现、数据模型、性能指标、埋点和迭代计划。
- 场景：情侣共同记录生活的 H5 应用。

## 2. 用户功能清单（MVP）

### 2.1 账号功能
- 账号密码注册
- 账号密码登录
- 退出登录

### 2.2 个人资料
- 编辑头像
- 编辑昵称
- 编辑生日
- 编辑个人简介

### 2.3 关系功能
- 生成邀请码邀请伴侣绑定
- 输入邀请码完成绑定
- 查看当前绑定状态
- 解绑关系（二次确认）

### 2.4 日记功能
- 发布日记（文字必填）
- 上传图片（最多 9 张）
- 查看时间线（按时间倒序）
- 编辑本人日记
- 删除本人日记

### 2.5 纪念日功能
- 新建纪念日（标题、日期、备注）
- 编辑纪念日
- 删除纪念日
- 首页展示纪念日倒计时

### 2.6 互动功能
- 日记点赞
- 日记评论
- 删除本人评论

### 2.7 隐私功能
- 未绑定时仅可见本人内容
- 绑定后双方可见共同内容
- 用户仅可编辑和删除自己发布的内容
